GP

Description

Global

The following options do check for variable/namespace existence. If already defined, we use that instance, otherwise we assign a new object literal to myApplication.

  • Option 1: var GP = GP || {};
  • Option 2 if(!GP) GP = {};
  • Option 3: var GP = GP = GP || {}
  • Option 4: GP || (GP = {});
  • Option 5: var GP = GP === undefined ? {} : GP;
